# ansible-training

Slides for Ansible Training at Suncorp. 

To see the slides:
```
git clone http://stash.int.corp.sun/scm/ANSI/ansible-training.git
cd ansible-training
python -m SimpleHTTPServer 8080
```
And then view http://localhost:8080/
